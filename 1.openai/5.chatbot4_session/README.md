# 챗봇 4 - 세션 관리

사용자별 세션을 분리하고 대화 요약 기능을 제공하는 챗봇입니다.

## 핵심 개념

- Flask 세션으로 사용자별 대화 분리
- 세션 삭제 기능
- GPT를 활용한 대화 요약

## 예제 목록

| 파일 | 설명 |
|------|------|
| `app5_session.py` | 기본 세션 관리 |
| `app6_session_delete.py` | 세션 삭제 기능 추가 |
| `app7_session_summary.py` | 대화 요약 기능 추가 |

## 실행

```bash
python app7_session_summary.py
```
