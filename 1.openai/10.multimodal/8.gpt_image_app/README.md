# gpt-image-1 활용 앱

OpenAI 최신 이미지 모델 `gpt-image-1`의 세 가지 기능을 단계별로 확장하는 웹앱입니다.

## 단계별 구성

| 폴더 | 기능 | 핵심 |
|------|------|------|
| `8.gpt_image_app/` | 1단계 — 이미지 생성 | 프롬프트 → 이미지 (`images.generate`) |
| `8.gpt_image_app2_inpaint/` | 2단계 — 특정 영역만 편집 | 영역 선택 → 그 부분만 재생성 (`images.edit` + 마스크) |
| `8.gpt_image_app3_consistency/` | 3단계 — 주인공 유지 변형 | 기준 이미지 참고 → 같은 피사체로 새 장면 (`images.edit`, 마스크 없음) |

## 핵심 개념

### gpt-image-1의 응답 형식
gpt-image-1은 이미지 URL이 아니라 **base64**(`response.data[0].b64_json`)로 결과를 줍니다.
그래서 이 앱들은 base64를 그대로 브라우저로 보내 `<img src="data:image/png;base64,...">`로 표시합니다.

### 영역 편집(인페인팅)과 마스크 — 2단계
- 마스크는 원본과 같은 크기의 RGBA 이미지입니다.
- **투명한(alpha=0) 영역 = 편집 대상**, 불투명한 영역 = 그대로 유지.
- 사용자가 캔버스에서 드래그한 사각형을 서버가 '투명 영역'으로 만들어 마스크를 생성합니다.
- 편집 결과를 다시 캔버스에 올려, 다른 영역을 이어서 편집할 수 있습니다.

### 일관성 유지 — 3단계
- 마스크 없이 `images.edit`에 기준 이미지를 넣으면, gpt-image-1이 그 이미지를
  **시각적 참고**로 삼아 같은 피사체(주인공)를 유지한 채 새 장면을 만듭니다.
- 2단계(마스크 O = 부분 수정)와 3단계(마스크 X = 참고 기반 전체 생성)의 차이가 핵심입니다.
- 참고 이미지를 여러 장 넣으면(`image=[...]`) 일관성이 더 좋아집니다.

## 이미지 모델 가격과 현황 (2026년 5월 기준)

이미지 생성은 토큰이 아니라 **이미지 1장당** 과금됩니다(아래는 1024×1024 기준).
크기·품질에 따라 달라지고 가격은 수시로 바뀌므로, 반드시
[OpenAI 공식 가격표](https://openai.com/api/pricing/)에서 최신 값을 확인하세요.

| 모델 | 상태 | Low | Medium | High |
|------|------|----:|-------:|-----:|
| `gpt-image-2` | 현 flagship | — | ~$0.04 (표준) | — |
| `gpt-image-1.5` | 현행 | ~$0.01 | ~$0.04–0.06 | ~$0.14–0.20 |
| `gpt-image-1` | **2026-10-23 종료 예정** | ~$0.011 | ~$0.042 | ~$0.167 |
| `gpt-image-1-mini` | 최저가 | ~$0.005 | ~$0.011 | ~$0.036 |
| `dall-e-2` / `dall-e-3` | **2026-05-12 API에서 제거됨** | — | — | — |

- `gpt-image-2`(현 flagship)의 전체 가격 범위는 이미지당 약 **$0.005 ~ $0.211** (크기·품질에 따라).
- gpt-image 계열은 내부적으로 토큰 단위로 과금됩니다(이미지 출력 **$40 / 1M 토큰**).
  1024×1024 기준 Low ≈ 272토큰, Medium ≈ 1,056토큰.

> ⚠️ 이 예제들은 `model='gpt-image-1'`을 사용합니다. 아직 동작하지만 **2026-10-23
> 종료 예정**이므로, 이후에는 `gpt-image-1.5` 또는 `gpt-image-2`로 모델명을 바꿔야 합니다.

## 실행

```bash
pip install flask openai pillow python-dotenv
cd 8.gpt_image_app          # 또는 8.gpt_image_app2_inpaint, 8.gpt_image_app3_consistency
python app.py
# 브라우저에서 http://localhost:5000 접속
```

API 키는 `1.openai/.env`(각 앱 폴더 기준 `../../.env`)에 설정합니다: `OPENAI_API_KEY=sk-...`
