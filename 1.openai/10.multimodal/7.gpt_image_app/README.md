# gpt-image 활용 앱

OpenAI 이미지 모델 `gpt-image-1.5`의 세 가지 기능을 단계별로 확장하는 웹앱입니다.

## 단계별 구성

| 폴더 | 기능 | 핵심 |
|------|------|------|
| `7.gpt_image_app/` | 1단계 — 이미지 생성 | 프롬프트 → 이미지 (`images.generate`) |
| `7.gpt_image_app2_inpaint/` | 2단계 — 특정 영역만 편집 | 영역 선택 → 그 부분만 재생성 (`images.edit` + 마스크) |
| `7.gpt_image_app3_consistency/` | 3단계 — 주인공 유지 변형 | 기준 이미지 참고 → 같은 피사체로 새 장면 (`images.edit`, 마스크 없음) |

## 핵심 개념

### gpt-image의 응답 형식
gpt-image 계열은 이미지 URL이 아니라 **base64**(`response.data[0].b64_json`)로 결과를 줍니다.
그래서 이 앱들은 base64를 그대로 브라우저로 보내 `<img src="data:image/png;base64,...">`로 표시합니다.

### 영역 편집(인페인팅)과 마스크 — 2단계
- 마스크는 원본과 같은 크기의 RGBA 이미지입니다.
- **투명한(alpha=0) 영역 = 편집 대상**, 불투명한 영역 = 그대로 유지.
- 사용자가 캔버스에서 드래그한 사각형을 서버가 '투명 영역'으로 만들어 마스크를 생성합니다.
- 편집 결과를 다시 캔버스에 올려, 다른 영역을 이어서 편집할 수 있습니다.

### 일관성 유지 — 3단계
- 마스크 없이 `images.edit`에 기준 이미지를 넣으면, gpt-image가 그 이미지를
  **시각적 참고**로 삼아 같은 피사체(주인공)를 유지한 채 새 장면을 만듭니다.
- 2단계(마스크 O = 부분 수정)와 3단계(마스크 X = 참고 기반 전체 생성)의 차이가 핵심입니다.
- 참고 이미지를 여러 장 넣으면(`image=[...]`) 일관성이 더 좋아집니다.

## gpt-image 모델 비교 (1 / 1.5 / 2)

gpt-image 계열은 세 버전이 있습니다. 이 예제는 `gpt-image-1.5`를 사용합니다.

| | `gpt-image-1` | `gpt-image-1.5` | `gpt-image-2` |
|--|--------------|-----------------|---------------|
| 출시 | 2025 | 2026 | 2026 |
| 상태 | **2026-10-23 종료 예정** | 현행 (이 예제가 사용) | 현 flagship |
| 품질 | 1세대 수준 | 우수 | 최고 |
| 텍스트 렌더링 | 보통 | 정확도 ~90–95% | 정확도 >99%, 48개 언어 |
| 최대 해상도 | 1024²급 | 1536px (3종 고정) | 4096²(4K)·커스텀·16:9 |
| 투명 배경 | — | **지원** | 미지원 |
| 속도 | — | 빠름(~1초) | ~2–3초 (레이아웃 reasoning 포함) |
| 특징 | 첫 GPT 이미지 모델 | 투명 배경·빠른 속도 | 그리기 전 레이아웃을 '계획'함 |

- `gpt-image-2`가 품질·텍스트·해상도에서 가장 앞서지만, **투명 배경이 필요하면 1.5**를 써야 합니다.
- 세 모델 모두 API 사용법(`images.generate`/`images.edit`, base64 응답)이 동일해
  **`model=` 값만 바꾸면 교체**됩니다.
- `gpt-image-1-mini` — 위 계열의 경량·저가 버전 (품질↓, 비용 최저).

## 이미지 모델 가격 (2026년 5월, 1024×1024 기준)

이미지 생성은 '이미지 1장당' 과금되며 크기·품질에 따라 달라집니다.
가격은 수시로 바뀌므로 [OpenAI 공식 가격표](https://openai.com/api/pricing/)에서 확인하세요.

| 모델 | Low | Medium | High |
|------|----:|-------:|-----:|
| `gpt-image-2` | ~$0.005 | (1.5보다 비쌈) | ~$0.211 |
| `gpt-image-1.5` | ~$0.01 | ~$0.04–0.06 | ~$0.133 |
| `gpt-image-1` | ~$0.011 | ~$0.042 | ~$0.167 |
| `gpt-image-1-mini` | ~$0.005 | ~$0.011 | ~$0.036 |

- Low 품질은 `gpt-image-2`가, Medium·High는 `gpt-image-1.5`가 더 저렴합니다.
- gpt-image 계열은 내부적으로 토큰 단위로 과금됩니다(이미지 출력 **$40 / 1M 토큰**).
  1024×1024 기준 Low ≈ 272토큰, Medium ≈ 1,056토큰.
- `dall-e-2`/`dall-e-3`는 2026-05-12자로 API에서 제거되어 더 이상 사용할 수 없습니다.

## 실행

```bash
pip install flask openai pillow python-dotenv
cd 7.gpt_image_app          # 또는 7.gpt_image_app2_inpaint, 7.gpt_image_app3_consistency
python app.py
# 브라우저에서 http://localhost:5000 접속
```

API 키는 `1.openai/.env`(각 앱 폴더 기준 `../../.env`)에 설정합니다: `OPENAI_API_KEY=sk-...`

> 참고: 이 예제는 `gpt-image-1.5`를 사용합니다. 더 최신인 `gpt-image-2`로 바꾸려면
> 각 `app.py`의 `model='gpt-image-1.5'`를 `'gpt-image-2'`로 바꾸면 됩니다(같은 계열).
