# 5.vscode — VSCode 에서 내 MCP 서버 사용 (준비 중 · Phase 2)

VSCode(1.102+) 와 GitHub Copilot **Agent Mode** 는 MCP 서버를 도구로 사용할 수 있다.
여기서는 `1.common/2.server_quickstart` 에서 만든 내 MCP 서버를 VSCode 에 연결한다.

## 예정 내용 (Phase 2)
- `.vscode/mcp.json` — 워크스페이스에 내 MCP 서버 등록 (stdio)
- Copilot Chat **Agent Mode** 에서 `add` / `word_count` 같은 내 도구 호출 워크스루
- (선택) 입력 변수/시크릿 처리, HTTP 서버 등록 예시
